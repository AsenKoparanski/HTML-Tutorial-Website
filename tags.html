HTML Tags

Heading Tags

Every document starts with a heading which is defined with the <h1-h6> tags. They have six levels of sizes which can be used:

<h1>Your heading number 1</h1>

<h2>Your heading number 2</h2>

<h3>Your heading number 3</h3>

<h4>Your heading number 4</h4>

<h5>Your heading number 5</h5>

<h6>Your heading number 6</h6>

Paragraph Tags

Paragraph tags are defined with the <p> tag and create a line of space under them:

<p>Your first paragraph</p>

<p>Your second paragraph</p>

<p>Your third paragraph</p>

Horizontal lines and line break tags

The break line and horizontal line tags are self-closing tags defined by <br/> and <hr/> They work the following way:

<p>I love learning about html.<br/>
I love learning about html.<br/></p>

<p>I love learning about html.<hr/>
I love learning about html.<hr/></p>

Image tags

We define images with the <img> tag. Now we will see how I have implemented the images in these examples:

<img src="Images/paragraphs.png" alt="TheImageFromTheSecondExample">

Hyperlink tags

Hyperlinks are defined with <a href="link or file name">Any desired name</a>. Here are two examples of a hyperlink:

<a href="www.youtube.com">Hyperlink to youtube</a>

<a href="C://Users/Documents/myhtmlfile.html">Will open the html file</a>

Span tags

Span tags are defined with <span> tag. Here are some span uses:

<p>Span<span style="color:red">Example</span>One </p>

<p><span style="color:green">Span</span><span style="color:blue">Example</span>One </p>

Division Tags

This is the so called <div> tag that plays a big role in grouping various other tags and dividing pages.
In this example each red rectangle represents a separate <div> tag containing content in it:

<div>< src="Images/divexample.png" </div>
