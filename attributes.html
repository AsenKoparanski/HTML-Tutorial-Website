Attributes

So far we have looked at most of the tags and how they can be used(<h1>,<p>,<div>, etc.). We used them simply by themselves but now we will be learning about attributes of tags which offer interesting options. An attribute defines the specifications of an HTML element and is placed inside it's opening tag. Each attribute consists of a name and value.

  • The name property defines what you want to set to the tag. A good example would be to change the colour in a <p> tag using style:
<p style="color:red">Hello World</p> <p style="background-color:red">Hello World</p>
  • The value is what you would like to change. In this case the style, we have used color and background-color. Here are some other examples where the attribute is different:

<p align="left">This is left aligned</p>

<p align="center">This is center aligned</p>

<p align="right">This is right aligned</p>

Essential Attributes

There are four essential attributes in HTML that can be used in almost all of the elements. Every one of their values needs to be quoted.

  • id - This can be used to uniquely identify any element with a specific id. This is very useful when wanting to make changes for example to only a specific paragraph(content overall).
<p id="example1">Using #id in CSS and applying style will only affect this paragraph</p>
  • class - This can be used to identify a number of elements with a specific style set in CSS. You would use this if you wanted for example to make all your paragraphs the same font/size/color/etc.
<p class="example2">Using .example2 in CSS you could change all this text here by applying color:red;. This will change the text to red on every paragraph containing the class "example2".</p>
  • title - This gives a tooltip containing the title you give on top of the element when you hover your cursor over it.
<p title="example3">Try hovering it over this text.</p>
  • style - This lets you use CSS styling within the HTML code like we have already seen in the previous examples.
<p style="color:red">Hello World</p>
