Formatting

Bold Text

Anything that is written in the <b></b> tags will appear bold.

Example of bold text here.

Italic text

Anything that is written in the <i></i> tags will appear Italic.

Example of Italic text here.

Underlined Text

Anything that is written in the <u></u> tags will appear underlined.

Example of underlined text here.

Strike through Text

Anything that is written in the <strike></strike> tags will appear with a dash through it - strike.

Example of strike text here.

Superscript Text

Anything that is written in the <sup></sup> tags will be displayed half a character's height above the other characters.

Example of supscript text here.

Subscript Text

Anything that is written in the <sub></sub> tags will be displayed half a character's height beneath the other characters.

Example of subscript text here.

Larger Text

Anything that is written in the <big></big> tags will appear bigger than the other characters.

Example of big text here.

Smaller Text

Anything that is written in the <small></small> tags will appear smaller than the other characters.
