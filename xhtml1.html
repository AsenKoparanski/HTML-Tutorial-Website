XHTML 1.0

XHTML 1.0 stands for Extensible HyperText Markup Language and is the first document type in the XHTML family. It is almost identical to the HTML 4.01 but it is just a cleaner and stricter version.

Differences in Document Structure

• The XHTML DOCTYPE is mandatory in order for the file to be of XHTML type.

• The xmlns attribute in <html> is mandatory.

• Certain tags like <html>, <head>, <title>, <body>

Differences in Elements

• All XHTML elements must be properly nested in one another. That would mean that the order of opening and closing them must be correct.

• All XHTML elements must always be closed. Even empty elements like <br/> and <hr/>.

• All XHTML elements must be in lower case.

• All XHTML elements must have one root element.

Differences in Attributes

• All XHTML attributes must be in lower case.

• All XHTML attributes must be quoted.

• Minimization of attributes is forbidden.

Convert your HTML file to XHTML

• Add the XHTML <!DOCTYPE> to the first line of every page.

• Add an xmlnx attribute to the <html> element of every page

• Change all element names to lowercase.

• Close all empty elements.

• Change all attribute names to lowercase.

• Quote all attribute values.
