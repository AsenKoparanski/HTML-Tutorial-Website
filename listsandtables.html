Lists

Unordered Lists

The unordered list is a collection of similar items that don't have specific order. It is created by using the <ul> tag and each item is specified by the <li> tag and marked by a bullet.

<ul><li>Chelsea</li>
<li>Newcastle</li>
<li>Tottenham</li>
<li>Arsenal</li>
</ul>

Ordered Lists

The ordered list comes in handy when you need to number your items instead of bulleting them. It is created by using the <ol> tag and each item is specified by the <li> tag incremented by one each time.

<ol><li>Chelsea</li>
<li>Newcastle</li>
<li>Tottenham</li>
<li>Arsenal</li>
</ol>

Definition Lists

The definition list offers you the option to list items like in a dictionary or encyclopedia. It is used mostly to present a list of terms with names and values. The <dl> tag defines the start of the list and </dl> the end of it. The <dt> and <dd> tags define respectively a term and it's definition.

<dl>
<dt><b>Chelsea</b>
<dd>Professional English Football Club</dd>
<dt><b>Arsenal</b>
<dd>Arsenal</dd>
</dl>


Tables

HTML offers it's users to arrange text, images, links, etc. into tables. All tables can be styled with CSS in any way a user chooses to. A basic table contains three main tags and those are the <table> which creates the table, the <tr> tag which creates a table row and the <td> tag which creates columns where information is stored. Here is an example of such a table:

<table> <tr>
<td>Chelsea</td>
<td>Newcastle</td>
<tr>
<td>Tottenham</td>
<td>Arsenal</td>
</tr> </table>

Another really important tag that changes the type of the table is the table heading <th> tag. Here is how a table would look like with all the forementioned tags:

<table> <tr>
<th>Football clubs</th>
</tr>
<tr>
<td>Chelsea,Newcastle</td>
</tr> </table>
