Blocks

Block elements

Block elements are ones that have a line appearing before and after them. Some examples of block elements are: <p>,<h1-h6>,<ul>,<hr/>.

Inline elements

Inline elements are ones that can appear within a sentence and don't need to appear on a new line. Some examples of inline elements are: <b>,<i>,<u>,<strong>,<strike>,<sup>,<sub>

Grouping elements

Two of the most important tags that are used in almost every HTML file are the <div> and <span> tags.

• The <div> tag is probably the most important one because it can group up different HTML tags inside it and also it can be styled with CSS. The div tag is used with block-level elements.
